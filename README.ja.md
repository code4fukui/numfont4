# numfont4
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

3x7ビットの数字フォントです。

## 機能
- 0-9の数字用のビット3x7フォントを提供します
- フォントパターンのの16進数表現をCSVファイルに含んでいます

## 要件
なし

## 使用方法
このプロジェクトは、フォントデータからCSVファイルを生成するための2つのJavaScriptファイルを提供しています:

1. `make_font4.js`: 数字0-9の3x7ビットのフォントパターンを含む `numfont4.csv` ファイルを生成します。
2. `make_font3.js`: 数字0-9の3x5ビットのフォントパターンを含む `numfont3.csv` ファイルを生成します。

生成されたCSVファイルを使用するには、プロジェクトにインポートし、必要に応じてフォントデータを使用することができます。
