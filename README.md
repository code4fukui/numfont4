# numfont4
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

3x7 bit number font.

## Features
- Provides a 3x7 bit font for numbers 0-9
- Includes CSV files with hexadecimal representations of the font patterns

## Requirements
None

## Usage
The project provides two JavaScript files for generating CSV files from the font data:

1. `make_font4.js`: Generates a `numfont4.csv` file with the 3x7 bit font patterns for numbers 0-9.
2. `make_font3.js`: Generates a `numfont3.csv` file with the 3x5 bit font patterns for numbers 0-9.

To use the generated CSV files, you can import them into your project and use the font data as needed.
